# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records for the RehabOS AI platform.

Each ADR captures a significant architectural decision, including:
- **Status:** Proposed, Accepted, Deprecated, Superseded
- **Context:** The forces at play and why the decision was needed
- **Decision:** The chosen approach
- **Consequences:** Trade-offs, risks, and impacts
- **Alternatives Considered:** What was rejected and why
